## Installation
First, prepare a Conda environment and activate it via

	conda create --name <MY_ENV_NAME> python=3.11
	conda activate <MY_ENV_NAME>

Next, install VOPP's requirements via

	pip install -r <VOPP_DIR>/requirements.txt

where ``<VOPP_DIR>`` is the root directory of VOPP.
		 
## Quick Start
    python problem.py --config configs/unc_navigation.yaml
